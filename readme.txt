Count visitors who has ad-blockers enabled

== Description ==

Count visitors who has ad-blockers enabled, display nice charts (last 24 hours, last 30 days), break down by country.

== Installation ==

1. Install from WordPress plugin directory or unpacking ZIP file into plugins directory;
2. Activate plugin;
  


= Requirements =
* PHP 5.2 or higher.
* WordPress v3.0.x or higher.
